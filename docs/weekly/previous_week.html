How to Generate This Report

  1. Run the weekly report script with a specific week:
    python scripts/generate_weekly_report.py --week-start YYYY-MM-DD --output previous
  2. Or archive the current week as previous when generating a new current week:
    python scripts/generate_weekly_report.py --week-start YYYY-MM-DD --archive-current

Note: The --archive-current flag will copy current_week.html to previous_week.html before generating a new current week report.
